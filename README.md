# 🌦️ Live Weather Notifier

A Python-based application that provides real-time weather updates and displays desktop notifications with emoji-enhanced weather conditions.

---

## 🚀 Features

- 🌡️ Real-time weather data using API
- 🔔 Desktop notifications
- 😊 Emoji-based weather conditions
- 🔁 Automatic updates at regular intervals
- 🛡️ Error handling for API and network issues
- 🌍 User input for city and API key

---

## 🛠️ Technologies Used

- Python
- Requests (API handling)
- Plyer (desktop notifications)
